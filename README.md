# 📊 Digital Experience Optimization — Portfolio

A portfolio of three end-to-end product & marketing case studies focused on **conversion funnel optimization**, **campaign landing page systems**, and **behavioral analytics-driven personalization**.

---

## 🗂 Projects

### [Project 1 — Conversion Funnel Optimization](./project1-funnel/index.html)
Mid-funnel strategy for a theme park & hotel booking flow.
- Funnel analysis (10K → 600 users across 4 stages)
- Root cause identification (biggest drop-off: Browse → Checkout at 62.5%)
- CTA, pricing, and checkout simplification recommendations
- A/B test: "Book Now" vs. "Reserve Your Spot Today" (+63% CVR)
- Outcome metrics: +63% CVR, −31% bounce rate, +$84K monthly revenue lift

### [Project 2 — Campaign Landing Page Optimization](./project2-landing/index.html)
High-converting campaign landing page system (Adobe-style).
- Page architecture: hero, CTA hierarchy, social proof
- Personalization by location, device type, and return behavior
- CMS workflow: marketing publishes without engineering
- Sequential A/B experiments on headline, layout, and imagery
- Outcome metrics: +47% CTR, 9.2% final CVR, −52% bounce rate

### [Project 3 — Behavioral Analytics + Personalization Engine](./project3-behavioral/index.html)
Behavior-driven personalization system inspired by Adobe Analytics.
- Tracks clicks, scroll depth, and session duration
- Key insights: mobile drop-off patterns, returning user value, session timing
- If/then personalization logic engine (new user, returning, mobile, high-intent)
- Generic vs. personalized experience A/B test (+97% CVR uplift)
- Outcome metrics: +97% engagement, +72% repeat visits, −38% bounce rate
